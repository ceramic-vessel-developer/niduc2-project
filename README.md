# niduc2-project
A project for niduc2 course 

## Project specification
Simulator with implemented ARQ algorithms to test network connectivity problems

## Usage
```
$ ./niduc.py
```

## Used algorithms
* Coding algorithms:
  * CRC
  * Parity bit
* Distortion algorithms:
  * Binary erasure channel
  * Symmetric binary channel

## Used technology
* Python
* NumPy
